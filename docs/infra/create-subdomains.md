In creating a subdomain, enter the following:

- Subdomain
   - If your main domain is `ventapilot.com` and you entered `app` subdomain, result will be `app.ventapilot.com`
- Select and check "Custom folder for subdomain" checkbox
   - Note: Do not check "Use public_html" directory checkbox as it will override and may cause extra complexity on setting up the main `.htaccess` of the main domain.
